# UI/UX and Functionality Updates (2025/12/26)

## バグ修正 (Bug Fixes)
- [ ] **リロードしないとコンパス等が動作しない不具合の修正** <!-- id: 0 -->
    - 原因調査: 初回ロード時にイベントリスナーが正しくアタッチされていない可能性。
- [ ] **2Dモードでのピン操作の改善** <!-- id: 1 -->
    - 現状: ピンを触るとリストが表示される。
    - 修正: ピンを触ったらその場所の情報（詳細）を表示する。

- [ ] **3Dモード: 不要なUIの非表示化** <!-- id: 3 -->
    - 3Dモードに入ったら、リストボタンとスライダーを消す。
- [ ] **3Dモード: 調整ボタンへのラベル追加** <!-- id: 4 -->

## 新規機能 (New Features)
